## Reading The Matrix

- `capabilities` describe the product's primary value surface, not every feature in its documentation.
- `integrations` list representative entry points or adjacent systems, not a complete integration catalog.
- Pricing model is a high-level billing shape and should not be read as a current price quote.
- Status means the product is listed as active in the current catalog; it does not replace a vendor availability check.

See each record in [`data/agents.json`](../data/agents.json) for first-party sources and verification dates.
